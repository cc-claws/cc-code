feat(peri-tui): shell 命令执行功能 - 支持 `!` 前缀直接执行系统命令

实现 TUI 层直接执行系统命令的能力，用户输入 `!` 前缀可绕过 Agent 直接执行 shell 命令。

修改内容：
- shell_command.rs 新增 ShellCommandRuntime 管理命令运行时状态
- shell_exec.rs 新增跨平台命令执行（cmd/bash）+ stdin 支持
- shell_history.rs 新增命令历史记录存储
- keyboard/normal_keys.rs 添加 `!` 前缀识别和快捷键处理
- main_ui/mod.rs 集成 shell 命令 UI 渲染和状态展示
- message_render.rs 添加 shell 命令结果消息渲染
- message_view/mod.rs 支持 shell 命令消息类型显示
- events.rs / thread_ops.rs / panel_ops.rs 集成事件处理
- theme.rs 添加 shell 命令专用主题样式
- .gitignore 添加本地测试文件忽略规则

特性/影响：
- 支持 `!git status`、`!cargo build` 等命令直接执行
- 命令结果记录在独立线程，不污染 Agent 对话历史
- 支持 stdin 交互（如需要确认的命令）
- 跨平台兼容（Windows cmd / Unix bash）
