多 Agent 快捷键速查
快捷键上下文作用
Ctrl+T主界面循环切换显示模式（normal → show teammates → ...）
Shift+↓主界面（team mode）展开 teammate 选择模式
↑/↓teammate 选择模式选择 teammate
Enterteammate 选中查看该 teammate 全文
fBackgroundTasksDialog前台化选中的 teammate
Esc查看 teammate返回 leader 视图
Ctrl+O主界面切换 detail mode（展开/折叠 agent 对话）
↓主界面打开 BackgroundTasksDialog
xBackgroundTasksDialogstop 选中 agent
Ctrl+XCtrl+KBackgroundTasksDialogstop 所有 agent
