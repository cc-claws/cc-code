### 依赖方向验证

对照 CLAUDE.md 中的依赖关系图 `peri-agent → peri-middlewares → peri-tui`，所有 Cargo.toml 依赖方向正确：

- `peri-agent`：无 workspace crate 依赖 ✓
- `peri-middlewares`：仅依赖 `peri-agent` + `peri-lsp` ✓
- `peri-tui`：依赖 `peri-agent` + `peri-middlewares` + `peri-lsp` + `peri-widgets` + `langfuse-client` ✓
- `peri-widgets` / `langfuse-client` / `peri-lsp`：无 workspace crate 依赖 ✓

> 注意：`peri-tui` 同时直接依赖 `peri-agent`（非仅通过 `peri-middlewares` 间接依赖），这在当前架构下合理（TUI 需要使用 core 类型如 `BaseMessage`/`ContentBlock`），但需留意未来重构时保持中层抽象不被绕过。

无万能模块（`utils.rs`/`common.rs`/`helpers.rs`/`misc.rs`）——良好 ✓
